# ✅ To-Do List 

A simple, clean, and efficient **Command Line Interface (CLI) To-Do List Manager** built using **Java** and **Object-Oriented Programming (OOP)** principles.
This project allows users to manage daily tasks directly from the terminal by adding, viewing, completing, and deleting tasks.

It is designed as a **beginner-friendly Java project** demonstrating core programming concepts such as collections, OOP design, and menu-driven CLI applications.

---

# 🚀 Features

✔️ Add new tasks
📋 View all tasks
✅ Mark tasks as completed
🗑️ Delete tasks
🧠 Object-Oriented structure
🖥️ Interactive CLI menu system

---

# 🛠️ Technologies Used

* ☕ **Java**
* 📦 **ArrayList (Java Collections Framework)**
* 🧱 **Object-Oriented Programming (OOP)**
* 💻 **Command Line Interface (CLI)**


---

# ⚙️ How the Application Works

The application runs in the terminal and provides a **menu-driven interface** that allows users to perform different operations on their task list.

### Menu Options

```
===== TO-DO LIST MENU =====

1. Add Task
2. View Tasks
3. Mark Task as Completed
4. Delete Task
5. Exit
```

Users can select a number to perform the corresponding operation.

# ▶️ How to Run the Project

---
